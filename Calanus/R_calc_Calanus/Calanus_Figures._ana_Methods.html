Calanus life from the CCA point of view

Вадим Хайтов

Первичные данные

  1. Исходная матрица обилий, которые дал мне Коля, содержала довольно много пустых ячеек. Я предположил, что это те случаи, когда на соответствующем горизонте глубины не было отмечено ни одной особи Calanus (любой стадии жизненного цикла). В принципе, это должно кодироваться, как ноль, но я поставил NA (все равно для ССА строчка матрицы состоящая только из нулей штука запрещенная).

  2. Не везде были измерены средовые показатели. В тех случаях, когда тот или иной показатель был измерен в предыдущую и в последующую съемку я присваивал пустой ячейке среднее значение между этими двумя. Если параметр не был измерен в две последовательные даты, то каждому из двух пропусков присваивалось среднее значение по двум ближайшим измереным. Если параметр не был измерен более чем в двух последовательных съемках (три и более), то пустой ячейке присваивалось значение NA.

  3. Поскольку параметры среды измерялись на горизонтах более дробных, чем производился лов сетью, то для получения сопоставимых глубин на станции D1 производилось следующее усреднение:

  • Для горизонта облова 0-10 м. усреднялись измерения на глубине 0, 5, 10 м

  • Для горизонта облова 10-25 м. усреднялись измерения на глубине 15, 25 м

  • Для горизонта облова 25 - дно усреднялись измерения на глубине 50, 65 м

  1. Параметры, измеренные на пяти горизонтах станции D3, приписывались каждому горизонту облова без усреднений.

  2. Для дальнейшего анализа все данные по обилию стадий были разделены на величину столба облова, то есть на величину разности между конечной и начальной глубиной.

  3. Для соблюдения требований ССА пришлось удалить все строчки, в которых было хотя бы одно значение NA. Внимание! Это привело к существенному сокращению количества проб, использованных для анализа. Если в полной матрице было 1620 пробы, то после всех описанных выше действий осталось 492 проб. В полном наборе проб были использованы данные 1998 - 2019 годов, а в сокращенном варианте материал 2012, 2016, 2009, 2010, 2018, 2019, 2015, 2017, 2013, 2011, 2014. Методика визуализации сезонных изменений связи популяции Calanus с параметрами среды

Был проведен канонический корреспндентный анализ (CCA) (ter Braak 1994, Legendre & Legendre 2012). В качестве зависимой матрицы была использована матица, столбцами которой (variables) были стадии жизненного цикла Calanus (Np, CI, CII, CIII, CIV, CV, CM, CF) строками матрицы были сборы, каждая строка - отдельная проба. В ячейках матрицы располагались значения обилия соответствующей стадии в данной пробе. Вторая матрица, использованная в CCA, характеризовала следующие параметры среды, описанные для каждой пробы: Глубина (срденее значение глубины для протяжки сетью), соленость, температура и концентрация хлорофилла, измеренные на тех же горизонтах глубины, где была взята проба. Дополнительно, в качестве дискретного предиктора, в модель CCA была включена точка взятия проб (две градации: D1 vs D3). В анализе были использованы только те пробы, для которых были проведены оценки всех элементов обеих матриц (всего 492 проб).

Для оценки статистической значимости модели CCA в целом, значимости отдельных constrained axes и конкретных предикторов был применен пермутационный метод (Borcard et al., 2011). Для оценки уровня значимости во всех анализах проводилось 9999 пермутаций.

Вычисления были проведены с использованием функций пакета vegan (Oksanen et al, 2019) для языка статистического программирования R 3.6.2 (R Core Team, 2019).
